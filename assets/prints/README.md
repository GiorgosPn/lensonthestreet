# assets/print

Εδώ ζει η φωτογραφία που εμφανίζεται στη σελίδα Prints.

- `demo.jpg` : η φωτογραφία του print (αντικατέστησέ τη με όποια θέλεις, ίδιο όνομα αρχείου).
  Προτεινόμενο: sRGB JPEG, μεγάλη πλευρά 1600 έως 2000 px, ποιότητα 80 (web preview, όχι το master αρχείο εκτύπωσης).
- Αν θέλεις άλλο όνομα ή περισσότερα prints, ενημέρωσε το `PRINT` object στο `js/prints.js`.
